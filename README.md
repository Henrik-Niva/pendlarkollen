# Pendlarkollen

Pendlarkollen är en kartbaserad kollektivtrafikapp som visar linjer, hållplatser och realtidsfordon för UL, SL och X-trafik.

## Tech stack

### Frontend
- React
- TypeScript
- Vite
- MapLibre GL

### Backend
- Python
- FastAPI
- GTFS static
- GTFS realtime
